Advanced Writes: Writing with Dynamically Loaded Filters

This tutorial presumes that you have completed the Advanced Read: Installing Dynamically Loaded Filters tutorial and have set up your environment to use your dynamically loaded filters.
DataPipe objects can be used to write using Dynamically loaded filters. This tutorial will be using the Zstd dynamic filter as an example.
The types.untyped.datapipe.properties.DynamicFilter property takes in an enumerated type ...datapipe.dynamic.Filter which is a hard-coded list of all listed registered filter plugins in HDF5.
zstdProperty = types.untyped.datapipe.properties.DynamicFilter(types.untyped.datapipe.dynamic.Filter.ZStandard);

Parameters

Some filter plugins allow for setting special configuration parameters to modify the filter's behavior. The DynamicFilter property type contains a modifiable parameters field which can be used to set your parameters. This is equivalent to setting the cd_values argument in HDF5. In the case of the Zstandard HDF5 plugin, the first (and only) array argument value indicates the compression level.
zstdProperty.parameters = 9; % compression level.

Multiple Filters

You can use multiple dynamic filters by concatenating multiple DynamicFilter properties together. They will be applied in order of the inserted array but after any default properties (i.e. compressionLevel, hasShuffle, etc.)
bitShuffleProperty = types.untyped.datapipe.properties.DynamicFilter(types.untyped.datapipe.dynamic.Filter.BitShuffle);
dynamicProperties = [bitShuffleProperty zstdProperty];

Writing

The DataPipe class takes in a keyword argument called filters which is an array of DynamicFilter objects.
Note: you cannot have compressionLevel enabled (anything other than -1) or hasShuffle enabled with extraFilters. If you still wish to use default Gzip compression, add it as filter property types.untyped.datapipe.properties.Compression .
% We're already compressing using zstd so we should disable
% compressionLevel (gzip).
dataPipe = types.untyped.DataPipe('data', rand(1, 10000), 'filters', dynamicProperties);
 
timeseries = types.core.TimeSeries('data', dataPipe, 'data_unit', 'data-unit');
 
nwbFile = NwbFile();
nwbFile.acquisition.set('ts', timeseries);
 
nwbExport(nwbFile, 'testdf.nwb');
 
The data is now compressed using Zstandard compression using a compression level of 9 and Bit Shuffled
